The Work Plan

A Work Plan is a comprehensive document that brings together all the key planning elements for a project. It serves as a central reference for the entire team.

Key Components of a Work Plan:

  • Work Activities and Outcomes: A summary of the project goals and the main steps required to achieve them.
  • Site Plan: A drawing of the work area, showing boundaries, existing structures, and where the new work will take place.
  • Tools and Equipment List: A list of all tools, machinery, and PPE required for the job. It's important to confirm the availability and condition of these items before work starts.
  • Job Safety Analysis (JSA): This is a critical part of the work plan. It details the specific hazards for each work activity and the control measures that will be used to keep everyone safe. It also assigns responsibility for implementing those controls.

The Materials List (or Take-Off)

A Materials List is a detailed inventory of every single component and material needed to complete the project. Creating this list is a process called a "material take-off," where you extract all the required items from the plans and specifications.

How to Create a Materials List:

  1. Review Plans and Specs: Carefully examine all drawings and the specification document.
  2. List Every Component: Write down every item needed, from large structural timbers to the smallest screws and brackets.
  3. Specify Details: For each item, list its specifications. This includes:
    • Material Type: e.g., ACQ Treated Pine, Melamine.
    • Dimensions (L, W, T): Length, Width, and Thickness.
    • Section / Specification: e.g., 150x75mm.
  4. Quantify: State the exact quantity required for each item (e.g., 8 lengths, 100 screws, 1m� of soil).
  5. Estimate Cost: Research the price for each item and calculate the total estimated cost. This is crucial for project budgeting.
